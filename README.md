# Book Identifier

Instantly identify any book — drag-select a cover or title on any webpage, or scan one from your camera or gallery — and get the title, author, Goodreads rating, plot summary, and what readers think.

## How to Use

1. **Install the Chrome Extension** *(coming real soon)*
   - Once available, install it from the Chrome Web Store
   - No account needed to get started

2. **Select a book on any webpage**
   - Drag your cursor over a book cover or book title on any page
   - A result panel will appear with the book details

3. **See the results instantly**
   - Title & author
   - Goodreads rating
   - Plot summary
   - What readers think
